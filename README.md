# ChronoIntel — Unified AI Workforce Analytics Platform
A master’s-level research and analytics project examining how **AI adoption reshapes technical roles, skills, and labor-market dynamics** over 2015–2025. The platform provides interactive dashboards, synthetic datasets, and actionable insights for educators, employers, and learners. [Live Project Link](https://shashidhar-reddy1905.github.io/Unified-AI-Workforce-Analytics-Platform/)

## Executive Summary

ChronoIntel explores the **impact of AI on five technical roles**:

- Data Analyst  
- Data Engineer  
- Dashboard Developer  
- IT Architect  
- Cybersecurity Analyst  

It tracks trends from 2015–2025 using synthetic datasets and visualizes adoption rates, automation risk, employment trends, salary growth, and skill demand.

## Features

- **Interactive Dashboard:** Explore trends by role, year, and region with KPIs and charts.  
- **Role-Focused Analysis:** Insights for the five key technical roles.  
- **Longitudinal Analysis:** Track trends over a decade.  
- **Multi-Metric Support:** AI adoption, automation risk, salaries, employment, skill demand.  
- **Ethical Data Use:** Uses high-quality synthetic data responsibly.
- 
## Key Insights

- AI adoption has increased consistently across all five roles.  
- Automation risk generally declines as upskilling grows.  
- Employment and salaries continue to grow despite AI adoption.  
- Skill demand rises in data engineering, cloud tooling, cybersecurity, visualization, and analytics.

## Data & Metrics

- **Roles:** Data Analyst, Data Engineer, Dashboard Developer, IT Architect, Cybersecurity Analyst  
- **Metrics:** AI Adoption Rate, Automation Risk, Salary, Employment Counts, Skill Demand Index  
- **Time Span:** 2015–2025  
- **Sources:** Synthetic datasets modeled on OECD, BLS, UNESCO, WEF indicators

## Architecture

1. **Frontend:** GitHub Pages website & embedded Power BI dashboard  
2. **Data Layer:** Synthetic CSV datasets  
3. **Semantic Layer:** Power BI data model with DAX KPIs  
4. **Visualization:** Interactive charts and filters by role, region, and year
